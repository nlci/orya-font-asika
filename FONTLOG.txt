FONTLOG
Asika - NLCI Oriya fonts
========================

This file provides detailed information on the NLCI Oriya fonts.
This information should be distributed along with the NLCI Oriya fonts and any derivative works.

Basic Font Information
----------------------

Pan Oriya font designed to support all the languages using the Oriya script.

ChangeLog
---------
